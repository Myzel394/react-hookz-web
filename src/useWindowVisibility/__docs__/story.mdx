# useWindowVisibility

Returns a boolean indicating whether the window is visible or not.

> **_This hook provides stable API, meaning returned methods does not change between renders_**

## Reference

```ts
function useWindowVisibility(ssrValue = true): () => boolean;
```

#### Importing

<ImportPath />

#### Arguments

- **ssrValue** _`boolean`_ _(default: `true`)_ - Value to return on server-side rendering.

#### Return

Boolean indicating whether the window is visible or not.
